== Description ==

CBIS search widget for WordPress made by Hultsfreds kommun.

This plugin uses CBIS to show a search widget.

All settings are in the widget.

Contact Jonas Hjalmarsson, Hultsfreds kommun on Twitter @hjalmarsson for questions.


== Installation ==

Download to your WordPress-installation or download and unzip manually to the plugins-folder.

Activate in Plugins section.

Drag and drop the widget to your widget-area in the Widgets section, all settings is placed there.
